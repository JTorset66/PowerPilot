# Chat Logs

Saved chat-memory files are written into this folder by `..\save-chat-memory.ps1`.

Suggested use:

- save important troubleshooting chats
- save build/install session summaries
- save decisions about telemetry, GPU behavior, and power-plan logic
- keep one short summary per session instead of huge raw dumps when possible

Reminder:

- take regular snapshots or commits alongside major session notes
