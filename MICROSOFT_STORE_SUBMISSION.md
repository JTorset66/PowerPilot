## Certification Notes

PowerPilot is a local Windows desktop utility. It does not require a user account, login, license key, server connection, or hidden test mode.

The app manages only PowerPilot-owned local Windows power plans, follows the Windows power mode selector, starts in the notification area after install when startup is enabled, and logs local battery/status data to local files only.

PowerPilot does not send telemetry or personal data over the network by itself. Battery logs, reports, or screenshots are shared only if the user manually copies or uploads them outside the app.

The About tab includes the installed version, Store developer identity, copyright, privacy summary, bundled manual/readme/license buttons, and local data boundaries.
